💡 Intent to Dorks

Describe what you're looking for in plain English. Claude Haiku generates precision Google dorking queries that match your intent.

Example: "S3 buckets with public access" → 8 targetted dorks

⚡ Concurrent Search

Execute all dorks in parallel (4 concurrent) via DuckDuckGo. No API keys, no rate limits, free. 15-second timeout per dork with graceful degradation.

🎯 Smart Dedup

Automatically deduplicate results by URL, retain highest confidence. Get clean, unique findings without noise.

🔄 Iterate Toggle

Generate mode: Fast dorks only (2s)
Iterate mode: Full loop—analyze results, refine intent, re-search (20s)

📊 Structured Output

JSON, CSV, or Markdown formats. Top 50 findings with URLs, titles, dorks used, confidence scores.

Architecture & Performance

Stack

Python 3.9+ Claude Haiku API httpx (async) DuckDuckGo (free) OpenClaw

Operation Workflow

[USER INPUT: "find vulnerable MCA targets"] ↓ [GENERATE: Haiku creates 8 precision dorks] ↓ [SEARCH: Concurrent execution (4 parallel)] ↓ [PARSE: Extract & deduplicate findings] ↓ [ANALYZE: Haiku suggests refined intent (iterate mode)] ↓ [REFINE: Generate 3 new dorks from refined intent] ↓ [SEARCH & MERGE: Final deduped results] ↓ [OUTPUT: JSON/CSV/Markdown with top 50 findings]

Cost & Performance

Generate mode (dorks only): ~2s, ~$0.00001 (1 Haiku call) Search execution (8 dorks): ~8s, $0 (DuckDuckGo free) Iterate mode (2 passes): ~15s, ~$0.0001 (2 Haiku calls) Full loop end-to-end: ~20s, ~$0.001
