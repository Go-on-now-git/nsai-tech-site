# NSAI Showcase Package

Sales and demo assets for NSAI — AI Operating System by Not So Holdings LLC.

## Contents

| File | Purpose |
|------|---------|
| `index.html` | Landing page / showcase site — open in browser, deploy anywhere |
| `pitch-deck.md` | One-pager pitch document — send to prospects, convert to PDF |
| `demo-script.md` | Live demo script for calls and screen recordings |

## Deployment

The landing page is a single self-contained HTML file. Deploy options:
- **GitHub Pages** — push to a repo, enable Pages
- **Netlify/Vercel** — drag and drop
- **Any web host** — just upload the file
